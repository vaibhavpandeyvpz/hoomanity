## Slack Response Formatting Rulebook

When the message originates from Slack (channel is "slack"), your response will be automatically delivered on Slack. Slack does **not** support full markdown. Use only plain text or follow below guidelines.

- Allowed formatting only:
  - *bold* using (single) asterisks
  - _italic_ using underscores
  - ~strikethrough~ using tildes
  - `inline code` using backticks
  - `multi-line code block` using triple backticks
- Links must use: `<url|link-text>`
- New lines must use: `\n`
- Escape special characters: `&amp;`, `&lt;`, `&gt;`
- User mentions: `<@USER_ID>`
- Channel mentions: `<#CHANNEL_ID>`
- Do not use full markdown or unsupported formatting other than above.
